# Release Notes - v2.0.1

Fecha: 2026-05-11

Resumen
- Correcciones y ajustes menores relacionados con includes en vistas y generación de PDFs.

Cambios

- Eliminación de includes relativos inválidos en varias vistas (`views/*.php`) que provocaban errores fatales.
- Centralización del inicio de sesión en `config/bootstrap.php` para evitar `Notice` que rompían la generación de PDFs.
- Ajustes en `pdf_cotizacion.php` y `pdf_receta.php` para un manejo más robusto de Dompdf y logs en `tmp/`.

Notas

- Si tu workflow de GitHub Actions (archivo `.github/workflows/release.yml`) crea releases al pushear tags `v*`, crea y empuja la etiqueta `v2.0.1` para activar automáticamente la creación del release.
